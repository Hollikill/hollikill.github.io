Easy Spanish Accents


This program will let you type the spanish accents you need much, much easier! As you know if you've tried it, typing letters such as: á, é, í, ó, ú, ñ, ¡, ¿, etc. can be very difficult! It seems like none of the solutions listed online work at all... but this one simple program can fix all that. Simply download it from this link. To use the program, press the control key [ ctrl ] and the tilde key [ ` ] at the same time. To then acess the help menu, type the h [ h ] key If you want to have it run all the time, allowing for even easier accents, simply run the run program (Accessible by pressing the windows key and typing run), type shell:common startup and press enter, and move spanish_accents.exe into the folder!
